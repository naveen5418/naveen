# Naveen Website

A modern, responsive website built with HTML, CSS, and JavaScript.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean and professional design with smooth animations
- **Navigation**: Sticky navbar with smooth scrolling
- **Hero Section**: Eye-catching banner with call-to-action
- **Services Section**: Showcasing your services in a grid layout
- **Contact Form**: Easy way for visitors to get in touch
- **Footer**: Complete with copyright information

## Sections

1. **Home** - Hero section with main call-to-action
2. **About** - Brief introduction to your business
3. **Services** - Display of offered services
4. **Contact** - Contact form for inquiries

## Files

- `index.html` - Main HTML structure
- `styles.css` - Styling and responsive design
- `script.js` - Interactive functionality and animations
- `README.md` - Project documentation

## Getting Started

1. Clone the repository
2. Open `index.html` in your web browser
3. Customize the content to match your needs

## Customization

### Colors
Edit the CSS variables in `styles.css` under `:root`:
- `--primary-color`: Main accent color
- `--secondary-color`: Secondary accent color
- `--dark-bg`: Dark background
- `--light-bg`: Light background

### Content
Edit the HTML in `index.html` to:
- Change section text
- Update service offerings
- Modify navigation links
